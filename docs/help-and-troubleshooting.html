Quick Help

Common issues and quick solutions to get you back on track

General Troubleshooting

Browser Issues

Common browser problems and solutions:

  • Page won't load: Check your internet connection and try refreshing the page
  • Tools not responding: Clear your browser cache and cookies
  • JavaScript errors: Ensure JavaScript is enabled in your browser settings
  • Slow performance: Close unnecessary tabs and restart your browser
  • Layout issues: Try zooming out (Ctrl/Cmd + -) or using a different browser

Mobile Device Issues

Mobile-specific troubleshooting:

  • Touch not working: Try tapping more firmly or using a different finger
  • Text too small: Pinch to zoom or use your browser's zoom feature
  • Keyboard issues: Rotate to landscape mode for better keyboard access
  • Slow loading: Check your mobile data connection or switch to WiFi
  • App-like experience: Add the site to your home screen for easier access

Performance Issues

Speed up your experience:

  • Slow loading: Close unnecessary browser tabs and applications
  • Laggy interactions: Restart your browser or device
  • Memory issues: Clear browser cache and temporary files
  • Network problems: Check your internet speed and connection stability
  • Old browser: Update to the latest version of your browser

Tool-Specific Issues

Find solutions for specific tools and their common problems

🔲 QR Code Generator Issues

Common QR Code problems:

  • QR code not generating: Check that you've entered content and selected a QR type
  • Download not working: Ensure pop-ups are allowed and try right-clicking the QR code to save
  • QR code not scanning: Try increasing the size or checking the content format
  • Library loading error: Refresh the page or try a different browser
  • Custom colors not working: Ensure you've selected both foreground and background colors

🔄 Unit Converter Issues

Unit conversion problems:

  • No result showing: Ensure you've entered a valid number and selected both units
  • Wrong conversion: Check that you've selected the correct unit category
  • Copy not working: Try clicking the copy button again or manually select and copy
  • Units not loading: Refresh the page or check your internet connection
  • Decimal precision: Results are rounded for display; increase precision if needed

🔤 Text Tools Issues

Text processing problems:

  • Case conversion not working: Ensure you've selected a case type and entered text
  • Word count incorrect: Check for hidden characters or formatting issues
  • Large text processing: Very large texts may take a moment to process
  • Special characters: Some special characters may not convert properly in certain cases
  • Copy functionality: Try selecting the text manually if the copy button doesn't work

🔐 Password Generator Issues

Password generation problems:

  • No password generated: Ensure at least one character set is selected
  • Password too weak: Increase length and include more character types
  • Copy not working: Try clicking the copy button again or manually select the password
  • Length slider not working: Try clicking on the slider or typing the value directly
  • Character sets not updating: Refresh the page if checkboxes aren't responding

🆔 Developer Tools Issues

UUID and Base64 problems:

  • UUID not generating: Ensure you've selected a UUID version and quantity
  • Base64 encoding error: Check that your input is valid text or Base64 string
  • Invalid Base64: Ensure the string is properly formatted Base64
  • Large data processing: Very large inputs may take time to process
  • Copy multiple UUIDs: Use the "Copy All" button for multiple UUIDs

Frequently Asked Questions

Answers to the most common questions

Yes! All tools on FreeThings.win are completely free to use. No registration, no hidden costs, no watermarks.

No! All processing happens in your browser. We don't store, transmit, or collect any of your data.

Most tools require an internet connection for initial loading. Once loaded, some tools may work offline.

Yes! All tools are free for any use, including commercial use. No attribution required.

Our tools are designed to be simple and effective. For advanced features, consider specialized software or services.
